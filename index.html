// About

Who's bluebo?

I've been scripting in Roblox Studio for over 5 years. Started the same way most people do, copy-pasting tutorials and wondering why nothing worked. Eventually figured out how to actually write clean, structured code and never looked back.

These days I specialize in modular systems, the kind of code that's easy to extend, easy to debug, and doesn't fall apart when a project grows. AI logic and combat are where I spend most of my time, but I'm comfortable taking on pretty much anything.

I keep projects small enough that I can stay hands-on. That means you're talking directly to the person writing your code, not some middleman. If something isn't right, we fix it.

I also run a YouTube channel where I post breakdowns of the systems I build — so if you want to see how I think about architecture before hiring me, that's a good place to start.
